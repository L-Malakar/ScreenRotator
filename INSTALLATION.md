# 🛠️ Installation & Setup Guide

This guide provides the technical steps required to set up the development environment, install dependencies, and compile **ScreenRotator Auto** into a standalone Windows executable.

---

## 1. Prerequisites

Before starting, ensure your system meets the following requirements:
* **Operating System:** Windows 10 or Windows 11.
* **Python Version:** [Python 3.8+](https://www.python.org/downloads/) (Ensure "Add Python to PATH" is checked during install).
* **System Permissions:** Administrator privileges are required to register global hotkeys and manipulate display settings.

---
